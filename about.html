About us

      OFOS is an online food ordering service which provides an easy way to order food from a restaurant online. Our service which provides an easy way to order food from a restaurant online. Our service can be adapted for use for any retaurant, all we need is a menu that will be displayed in our tabular menu format, creating an intuitive process for ordering menu items. We also provide customer account serviecs which make it easier than ever to order food. A user just needs to create an account which stores a user's name, address, and credit card information. This information is then accessed when an order is placed to expedite the ordering process. The address is loaded if a user orders food for delivery. The credit card information is used to pay for the order using the online payment services provided by Charge.com.
